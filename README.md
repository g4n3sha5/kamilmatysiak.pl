
## Getting Started

It's my personal portfolio page, made with Next.JS and Bootstrap

## Technologies 
Project made with Bootstrap, Next.JS, CSS3, HTML5.
### How to run it?
It's live here:

[kamilmatysiak.pl](https://www.kamilmatysiak.pl/)

### Locally
Install dependencies:
```
npm install
```

Run developer server:

```
npm run dev
```
